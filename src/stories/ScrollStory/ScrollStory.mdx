# ScrollStory

A scrollytelling component that displays media (images, videos, AI2HTML graphics) alongside scroll-triggered text. Media transitions smoothly as the user scrolls through the content.

## Props

### `data.slides` (required)  
**Type:** `Array<{ media: string, text?: string, altText?: string }>`  
A list of slides, each containing a `media` URL (image, YouTube, Vimeo, or MP4), optional `text` (HTML allowed), and optional `altText` for images.

---

### `data.imgFit`  
**Type:** `"cover" | "contain" | "fill"`  
**Default:** `"cover"`  
Sets how the media is resized inside its container (`object-fit`).

---

### `data.containerWidth`  
**Type:** `"inline" | "edge-to-edge" | "bleed"`  
**Default:** `"inline"`  
Controls the width of the media container.  
- `inline`: uses 56.2% width and centers the media  
- `bleed`: respects content layout  
- `edge-to-edge`: stretches to full viewport width  

---

### `data.borderRadius`  
**Type:** `string`  
**Default:** `"0"`  
Sets the `border-radius` of the media element (e.g., `"8px"` or `"1rem"`).

---

### `data.fixedImgHeight`  
**Type:** `string`  
If provided (e.g. `"500px"`), sets a fixed height for all media. Otherwise it defaults to `"500px"` or `"100vh"` for `edge-to-edge`.

---

### `data.fontSize`  
**Type:** `"sm" | "md" | "lg"`  
**Default:** `"md"`  
Adjusts the font size of the scroll-triggered text blocks.

---

### `data.videoAutoplay`  
**Type:** `boolean`  
**Default:** `true`  
If `true`, videos autoplay when active.

---

### `data.videoLoop`  
**Type:** `boolean`  
**Default:** `true`  
If `true`, videos will loop.

---

### `data.videoControls`  
**Type:** `boolean`  
**Default:** `false`  
If `true`, native video player controls will be shown.

---

### `data.videoMute`  
**Type:** `boolean`  
**Default:** `true`  
If `true`, videos will start muted.

---


## Usage

```jsx
import { ScrollStory } from '@times/graphics-components';

const data = {
  imgFit: "cover",
  containerWidth: "edge-to-edge",
  borderRadius: "0",
  fixedImgHeight: "500px",
  fontType: "sans-serif",
  fontSize: "md",
  videoAutoplay: true,
  videoControls: false,
  videoLoop: true,
  videoMute: true,
  slides: [
    {
      media: "https://picsum.photos/id/1015/1200/800",
      text: "This is a slide with an image."
    },
    {
      media: "https://www.youtube.com/watch?v=wDchsz8nmbo",
      text: "This is a slide with a YouTube video."
    },
    {
      media: "https://www.barrons.com/asset/barrons/videos/asset-uploader-a9e12c89-5df7-433d-98aa-868c701b078d/C0586.mp4",
      text: "This is a slide with an MP4 video."
    }
  ]
};

<ScrollStory data={data} />
