# Palettes

The palettes are a group of colours that are used tied to a single product but are part of the growing list of colours that represent all the brand colours available for use in The Times.

These colours are free to use but should be critically evaluated for the benefit of their use within a design or product.

---

## Generating colours

The core palettes are defined using the same lightness code for each colour tone, therefore palettes with the same number have the same perceived lightness and WCAG contrast ratio.

Creating accessible light and dark themes is easier and quicker knowing which palette combinations meet the accessibility requirements.

## Usage Guidelines

- **Accessibility First**: All colour combinations should meet WCAG AA contrast requirements (minimum 4.5:1 ratio)
- **Consistent Lightness**: Colours within the same palette weight (100, 200, etc.) have consistent perceived lightness
- **Theme Flexibility**: The systematic approach makes it easier to create both light and dark themes
- **Brand Compliance**: These colours represent The Times brand palette and should be used thoughtfully

## Tools for Color Generation

The following online tools can help generate accessible color palettes:

- **Leonardo Accessible Palettes**: For creating accessible color systems
- **Geenes Color Tool**: For generating and testing color combinations

Remember to always validate color combinations for accessibility compliance before implementation.
