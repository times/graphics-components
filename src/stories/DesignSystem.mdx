## Usage Guidelines

### Color Usage
- **Accessibility First**: All colour combinations should meet WCAG AA contrast requirements (minimum 4.5:1 ratio)
- **Channel Consistency**: Use appropriate channel colors for content categorization
- **Semantic Colors**: Use semantic colors consistently for error, warning, success, and info states
- **Theme Support**: Utilize ink colors for proper light/dark theme implementation

### Typography
- **Times Modern**: Primary typeface for editorial content and headlines
- **Inter**: Secondary typeface for UI elements and body text
- **Hierarchy**: Use consistent font size and weight combinations to establish clear visual hierarchy
- **Editorial Styles**: Leverage predefined editorial display and headline styles for consistency

### Spacing
- **Consistent Scale**: Use the defined spacing scale (0-32) for consistent layouts
- **8-Point Grid**: Base system follows 8-point grid principles
- **Component Spacing**: Apply consistent internal and external spacing to components

### Shadows
- **Depth Hierarchy**: Use shadows to indicate different levels of elevation
- **Consistency**: Apply shadows consistently across similar components
- **Performance**: Use shadows sparingly to maintain good performance

### Gradients
- **Overlay Usage**: Use overlay gradients to ensure text readability over images
- **Theme Awareness**: Choose appropriate gradient variants for light/dark themes

Remember to always validate your design choices against accessibility standards and brand guidelines.
