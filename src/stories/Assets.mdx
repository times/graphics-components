### Logo Usage Guidelines

**DO:**
- Maintain minimum size requirements for readability
- Use appropriate theme variants for background context
- Preserve aspect ratios when scaling
- Ensure adequate clear space around logos
- Use SVG format when possible for scalability

**DON'T:**
- Stretch or distort logo proportions
- Use light mode logos on dark backgrounds
- Place logos on busy or low-contrast backgrounds
- Modify colors or add effects to brand assets
- Use logos smaller than minimum specifications

### File Formats Available

- **SVG**: Scalable vector format (recommended)
- **PNG**: High-resolution raster format
- **JPG**: Compressed format for photography contexts
