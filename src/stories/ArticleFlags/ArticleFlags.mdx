# ArticleFlags

A component that renders one or multiple article flags such as **LIVE**, **BREAKING**, **EXCLUSIVE**, **UPDATED**, **SPONSORED**, **NEW**, and optionally a **Long Read** badge.

<Meta title="Furniture Components/ArticleFlags" component={ArticleFlags} />

## Overview

The `ArticleFlags` component displays visual labels for articles with metadata such as status, exclusivity, or timeliness. It also supports an optional "Long Read" indicator.

- üö® **LIVE/BREAKING**: Includes an animated pulsing square.

- ‚è± **Expiry Support**: Flags with an `expiryTime` automatically disappear when outdated.
