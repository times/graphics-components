# ImageSlider

A component for comparing two images using an interactive slider. Commonly used for before/after visual comparisons.

- ‚ÜîÔ∏è **Draggable Slider**: Interactive handle reveals either side of the image.
- üñº **Image Comparison**: Accepts `beforeImage` and `afterImage` props.
- üè∑ **Custom Labels**: Show optional "Before" and "After" tags.

- üì± **Responsive**: Works on desktop and mobile.

## Props

### `beforeImage` (required)  
**Type:** `string`  
URL of the "before" image.

---

### `afterImage` (required)  
**Type:** `string`  
URL of the "after" image.

---

### `beforeLabel`  
**Type:** `string`  
Default: `'Before'`  
Text label shown on the left side.

---

### `afterLabel`  
**Type:** `string`  
Default: `'After'`  
Text label shown on the right side.

---

### `photoCredits`  
**Type:** `{ before?: string; after?: string }`  
Optional object to display source credits below the component.

---

## Usage

```jsx
import { ImageSlider } from '@times/graphics-components';

<ImageSlider
  beforeImage="https://placehold.co/800x450?text=Before"
  afterImage="https://placehold.co/800x450?text=After"
  beforeLabel="Before Renovation"
  afterLabel="After Renovation"
  photoCredits={{
    before: 'NASA (2020)',
    after: 'NOAA (2024)'
  }}
/>
